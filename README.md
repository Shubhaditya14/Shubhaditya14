## What I've Built

**[Transformer Inference Accelerator](https://github.com/Shubhaditya14)** — FPGA-based hardware accelerator for transformer inference on Artix-7. INT8 post-training quantization pipeline, UART host interface, custom memory layout for weight tiling. Built as a capstone with a PCB team; I own the ML side end-to-end.

**[NietzscheGPT](https://github.com/Shubhaditya14/nietzsche-gpt)** — Character-level GPT trained on the complete works of Nietzsche. Then extended into a Tiny Inference Engine with KV-cache and INT8 PTQ. Followed Karpathy's nanoGPT then went further.

**[ScratchGPT](https://github.com/Shubhaditya14/ScratchGPT)** — GPT-2 implemented from scratch. No Hugging Face, no shortcuts.

**[Distributed Training Orchestrator](https://github.com/Shubhaditya14/Distributed)** — Master-worker architecture over gRPC. Fault tolerance, checkpoint recovery, all-reduce semantics. Built to understand what frameworks like PyTorch Distributed are actually doing.

**[Federated Learning Aggregation Server](https://github.com/Shubhaditya14)** — FL system designed for hospital deployment. Hadoop HDFS + MapReduce + Apache Hive for gradient aggregation, FastAPI for the coordination layer. Local simulation as showcase.

**[Music Recognition](https://github.com/Shubhaditya14/music-recognition)** — Shazam clone built from scratch. Spectrogram fingerprinting, hash-based matching.
